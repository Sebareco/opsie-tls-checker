# Analizador de Seguridad TLS

Proyecto híbrido con **Bun** (Backend) y **React + TS** (Frontend).

## Requisitos
- [Bun](https://bun.sh) instalado.

## Instalación
1. Clonar el repo.
2. Ejecutar `bun install` en la raíz.
3. Ejecutar `cd frontend && bun install`.

## Uso
- **Desarrollo:** `bun dev:ui` y `bun dev:server` (en terminales separadas).
- **Producción:** `bun run build:ui` y luego `bun start`.
